# 一、Tensor 与 NumPy 转换

## 1.Tensor 转 NumPy

```python
data_tensor = torch.tensor([2, 3, 4])

# 转换为 numpy
data_numpy = data_tensor.numpy()

print(type(data_tensor))
print(type(data_numpy))
```

## 2.避免共享内存

默认情况下：

```python
tensor.numpy()
```

得到的 ndarray 和 tensor 共享内存。

解决方法：

```python
data_numpy = data_tensor.numpy().copy()
```

这样修改其中一个不会影响另一个。

---

## 3.NumPy 转 Tensor

### 方法1：from_numpy（共享内存）

```python
data_numpy = np.array([2, 3, 4])

data_tensor = torch.from_numpy(data_numpy)
```

特点：

* 共享内存
* 修改一个另一个也会变

---

### 方法2：torch.tensor（不共享内存）

```python
data_numpy = np.array([2, 3, 4])

data_tensor = torch.tensor(data_numpy)
```

特点：

* 不共享内存
* 更安全

# 三、点乘（Hadamard）

点乘：

* 相同位置元素相乘
* 两个张量 shape 必须一致

---

## 示例

```python

data2 = torch.tensor([[5, 6], [7, 8]])

data = torch.mul(data1, data2)

```

结果：

```python
tensor([[ 5, 12],
        [21, 32]])
```

# 七、激活函数

激活函数作用：

给网络增加非线性能力。

如果没有激活函数：

整个神经网络本质上还是线性模型。

---

# 八、Sigmoid 激活函数

---

## 1.特点

输出范围：

```text
(0,1)
```

适用于：

* 二分类输出层

---

## 2.缺点

### 梯度消失

当输入：

```text
<-6 或 >6
```

导数接近 0。

深层网络训练困难。

---

## 3.问题

* 非 0 中心
* 收敛慢
* 深层网络容易梯度消失

---

# 九、Tanh 激活函数

---

## 1.特点

输出范围：

```text
(-1,1)
```

比 sigmoid 更好。

因为：

* 以 0 为中心
* 梯度更大

---

## 2.缺点

仍然会梯度消失。

---

# 十、ReLU 激活函数

---

## 1.公式

ReLU(x)=\max(0,x)

---

## 2.优点

* 计算简单
* 训练快
* 缓解梯度消失
* 当前最常用

---

## 3.缺点

Dead ReLU：

部分神经元永远输出 0。

---

# 十一、Softmax 激活函数

---

## 1.作用

多分类概率输出。

输出：

* 每个值在 0~1
* 总和为 1

---

## 2.示例

```python
scores = torch.tensor([0.2, 0.02, 0.15])

probabilities = torch.softmax(scores, dim=0)
```

---

# 十七、完整网络结构

```python
class Model(nn.Module):

    def __init__(self):
        super(Model, self).__init__()

        self.linear1 = nn.Linear(3, 3)

        self.linear2 = nn.Linear(3, 2)

        self.out = nn.Linear(2, 2)

    def forward(self, x):

        x = self.linear1(x)
        x = torch.sigmoid(x)

        x = self.linear2(x)
        x = torch.relu(x)

        x = self.out(x)

        x = torch.softmax(x, dim=-1)

        return x
```
