## 一、激活函数

激活函数的作用：为神经网络引入**非线性因素**，使网络能够学习和拟合复杂的模式。如果没有激活函数，多层神经网络等价于一个单层线性变换。

### 1.1 Sigmoid 函数

**公式**：

$$\sigma(x) = \frac{1}{1 + e^{-x}}$$

**特性**：
- 输出范围：(0, 1)，适合表示概率
- x=0 时，输出 0.5
- 导数公式：$f'(x) = f(x)(1 - f(x))$，导数最大值 0.25（在 x=0 处）
- **优点**：适合二分类问题的输出层
- **缺点**：当 |x| 较大时梯度接近 0，导致**梯度消失**问题

```python
import torch

# Sigmoid 函数计算
x = torch.linspace(-20, 20, 1000)
y = torch.sigmoid(x)

# Sigmoid 梯度计算
x = torch.linspace(-20, 20, 1000, requires_grad=True)
torch.sigmoid(x).sum().backward()
# x.grad 即为各点的梯度值
```

### 1.2 Tanh 函数

**公式**：

$$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$

**特性**：
- 输出范围：(-1, 1)
- **零中心化**：通过原点 (0,0)，比 Sigmoid 收敛更快
- 导数公式：$f'(x) = 1 - \tanh^2(x)$，导数最大值 1（在 x=0 处）
- **优点**：比 Sigmoid 梯度更强，收敛速度更快
- **缺点**：仍存在梯度消失问题，常用于隐藏层

```python
x = torch.linspace(-20, 20, 1000)
y = torch.tanh(x)

# Tanh 梯度计算
x = torch.linspace(-20, 20, 1000, requires_grad=True)
torch.tanh(x).sum().backward()
```

### 1.3 ReLU 函数

**公式**：

$$\text{ReLU}(x) = \max(0, x)$$

**特性**：
- 输出范围：[0, +∞)
- x < 0 时输出 0，x ≥ 0 时输出 x（线性）
- 导数：x > 0 时为 1，x < 0 时为 0
- **优点**：计算简单，有效缓解梯度消失问题
- **缺点**：存在**神经元死亡**问题（负区间梯度永远为 0）
- 现代深度学习**最常用**的隐藏层激活函数

```python
x = torch.linspace(-20, 20, 1000)
y = torch.relu(x)  # 负数归零，正数保持不变

# ReLU 梯度计算
x = torch.linspace(-20, 20, 1000, requires_grad=True)
torch.relu(x).sum().backward()
```

### 1.4 Softmax 函数

**公式**：

$$\text{softmax}(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$$

**特性**：
- 将输出转换为**概率分布**，所有输出之和为 1
- 主要用于**多分类问题**的输出层
- 会放大最大值的效应

```python
scores = torch.tensor([[0.3, 0.24, 5, -3.1],
                       [-0.1, 0.56, 2.4, 0.35]])

# dim=1: 按行计算softmax（常用），每个样本各类别概率和为1
prob = torch.softmax(scores, dim=1)
# 第一行: [6.2e-03, 5.8e-03, 9.87e-01, 1.1e-04]
# 最大值5.0对应概率高达0.987，显著放大

# dim=0: 按列计算softmax，每个类别在各样本间的概率分布
prob = torch.softmax(scores, dim=0)
```

### 1.5 激活函数选择总结

| 激活函数 | 输出范围 | 适用位置 | 优点 | 缺点 |
|---------|---------|---------|------|------|
| Sigmoid | (0,1) | 二分类输出层 | 概率解释 | 梯度消失 |
| Tanh | (-1,1) | 隐藏层 | 零中心化 | 梯度消失 |
| ReLU | [0,+∞) | 隐藏层（首选） | 计算快、缓解梯度消失 | 神经元死亡 |
| Softmax | 概率和为1 | 多分类输出层 | 概率分布 | 放大最大值 |

## 二、参数初始化

参数初始化对神经网络训练至关重要。不恰当的初始化会导致梯度消失或梯度爆炸。

### 2.1 初始化方法选择原则

**从激活函数角度**：
- **ReLU 系列**（ReLU、Leaky ReLU）→ 推荐 **Kaiming 初始化**
- **S 型函数**（Tanh、Sigmoid）→ 推荐 **Xavier 初始化**

**从网络深度角度**：
- **浅层网络**（<10 层）：可使用各种初始化方法
- **深层网络**（≥10 层）：推荐 Kaiming 或 Xavier

### 2.2 各种初始化方法

```python
import torch.nn as nn

linear = nn.Linear(5, 3)

# 1. 均匀分布初始化 U(0,1)
nn.init.uniform_(linear.weight)

# 2. 正态分布初始化 N(0,1)
nn.init.normal_(linear.weight, mean=0, std=1)

# 3. 全零初始化（不推荐用于训练，会导致对称性问题）
nn.init.zeros_(linear.weight)

# 4. 全一初始化（不推荐，可能导致梯度爆炸）
nn.init.ones_(linear.weight)

# 5. 常数初始化
nn.init.constant_(linear.weight, val=10)

# 6. Kaiming 初始化（适合 ReLU）
nn.init.kaiming_uniform_(linear.weight, nonlinearity='relu')
nn.init.kaiming_normal_(linear.weight, nonlinearity='relu')
# 数学原理: 正态分布标准差 = √(2/fan_in)

# 7. Xavier 初始化（适合 Tanh/Sigmoid）
nn.init.xavier_uniform_(linear.weight)
nn.init.xavier_normal_(linear.weight)
# 数学原理: 正态分布标准差 = √(2/(fan_in + fan_out))
```

### 2.3 初始化方法对比

| 方法 | 适用激活函数 | 优点 | 缺点 |
|------|------------|------|------|
| 均匀/正态分布 | 通用 | 简单、打破对称性 | 范围需手动选择 |
| 全零/全一 | 仅调试 | 实现简单 | 对称性问题 |
| Kaiming | ReLU 系列 | 避免梯度消失/爆炸 | 对 S 型函数效果一般 |
| Xavier | Tanh/Sigmoid | 保持输入输出方差一致 | 对 ReLU 效果欠佳 |

## 三、搭建神经网络

### 3.1 基本流程

搭建神经网络的标准步骤：
1. **定义类**：继承 `nn.Module`
2. **`__init__` 方法**：定义网络层结构和参数初始化
3. **`forward` 方法**：定义前向传播过程和激活函数

### 3.2 代码示例

```python
import torch
import torch.nn as nn

class MyANN(nn.Module):
    def __init__(self):
        super().__init__()
        # 定义网络各层
        self.linear1 = nn.Linear(3, 3)   # 隐藏层1: 3→3
        self.linear2 = nn.Linear(3, 2)   # 隐藏层2: 3→2
        self.output = nn.Linear(2, 2)    # 输出层: 2→2

        # 参数初始化
        nn.init.xavier_normal_(self.linear1.weight)  # Xavier（配合Sigmoid）
        nn.init.zeros_(self.linear1.bias)
        nn.init.kaiming_normal_(self.linear2.weight) # Kaiming（配合ReLU）
        nn.init.zeros_(self.linear2.bias)

    def forward(self, x):
        x = torch.sigmoid(self.linear1(x))   # 隐藏层1: 线性+Sigmoid
        x = torch.relu(self.linear2(x))       # 隐藏层2: 线性+ReLU
        x = torch.softmax(self.output(x), dim=-1)  # 输出层: 线性+Softmax
        return x

# 使用模型
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = MyANN().to(device)
data = torch.randn(10, 3).to(device)
output = model(data)
```

**重要提示**：模型和数据必须在同一设备上（CPU 或 GPU），否则报 `RuntimeError: Expected all tensors to be on the same device`。

### 3.3 查看模型信息

```python
from torchsummary import summary

# 打印模型结构摘要
summary(model, input_size=(3,), batch_size=1)

# 遍历所有参数
for name, param in model.named_parameters():
    print(f"参数名称: {name}, 参数值: {param}")
```

## 四、损失函数

### 4.1 分类问题的损失函数

#### 多分类：CrossEntropyLoss

- 内置 Softmax + 交叉熵计算
- 真实值可以是**类别索引**（Long 类型）或 **one-hot 编码**（float 类型）
- 预测值需要是 float 类型并设置 `requires_grad=True`

```python
import torch.nn as nn

# 真实值 - one-hot 编码
y_true = torch.tensor([[0,1,0], [0,0,1]], dtype=torch.float32)
# 预测值
y_pred = torch.tensor([[0.1,0.6,0.3], [0.7,0.1,0.2]], requires_grad=True)

loss_obj = nn.CrossEntropyLoss()
loss = loss_obj(y_pred, y_true)
```

> **注意**：使用 `CrossEntropyLoss` 时，模型输出层**不要**再加 Softmax，因为该损失函数内部已包含。

#### 二分类：BCELoss

```python
y_true = torch.tensor([0, 1, 0], dtype=torch.float32)
y_pred = torch.tensor([0.69, 0.55, 0.25], requires_grad=True)

loss = nn.BCELoss()(y_pred, y_true)
```

### 4.2 回归问题的损失函数

| 损失函数 | 别名 | 特点 | 使用频率 |
|---------|------|------|---------|
| L1Loss | MAE | 曲线不光滑，容易错过极值点 | 少用，常做正则化项 |
| MSELoss | MSE/L2 | 对损失求平方，容易梯度爆炸 | 较少，常做正则化项 |
| SmoothL1Loss | Huber | 结合 L1 和 L2 的优点 | **常用** |

```python
# L1损失 (MAE)
loss = nn.L1Loss()(y_pred, y_true)

# L2损失 (MSE)
loss = nn.MSELoss()(y_pred, y_true)

# Smooth L1损失（推荐）
loss = nn.SmoothL1Loss()(y_pred, y_true)
```

Smooth L1 特点：误差较小时用平方损失（梯度平滑），误差较大时用线性损失（避免梯度爆炸）。

## 五、梯度下降优化器

### 5.1 四种主流优化器

#### Momentum（动量法）
- **优化对象**：梯度
- **作用**：平缓梯度、帮助跳出鞍点
- **关键参数**：`momentum=0.9`

```python
optimizer = torch.optim.SGD([w], lr=0.1, momentum=0.9)
```

#### Adagrad
- **优化对象**：学习率
- **特点**：自适应调整每个参数的学习率
- **缺点**：学习率可能过快衰减，导致训练停滞

```python
optimizer = torch.optim.Adagrad([w], lr=0.1)
```

#### RMSprop
- **优化对象**：学习率
- **特点**：在 Adagrad 基础上增加 β 系数，缓解学习率衰减过快问题
- **关键参数**：`alpha=0.9`（平滑常数）

```python
optimizer = torch.optim.RMSprop([w], lr=0.1, alpha=0.9)
```

#### Adam（推荐）
- **优化对象**：梯度 + 学习率
- **特点**：结合 Momentum 和 RMSprop 的优势
- **关键参数**：`betas=(0.9, 0.99)`
- **最常用**的优化器

```python
optimizer = torch.optim.Adam([w], lr=0.1, betas=(0.9, 0.99))
```

### 5.2 优化器标准使用流程

```python
# 1. 创建优化器
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)

for epoch in range(epochs):
    for x, y in dataloader:
        # 2. 前向传播
        y_pred = model(x)
        loss = criterion(y_pred, y)

        # 3. 梯度清零（必须！因为 PyTorch 默认累积梯度）
        optimizer.zero_grad()
        # 4. 反向传播
        loss.backward()
        # 5. 更新参数
        optimizer.step()
```

## 六、学习率衰减策略

学习率衰减可以在训练后期更精细地调整参数，提高模型精度。

### 6.1 三种常用策略

```python
optimizer = torch.optim.SGD([w], lr=0.1, momentum=0.9)

# 1. StepLR: 等间隔衰减
# 每隔 step_size 个 epoch，学习率乘以 gamma
scheduler = torch.optim.lr_scheduler.StepLR(optimizer, step_size=50, gamma=0.5)

# 2. MultiStepLR: 指定间隔衰减
# 在指定的 milestone 处衰减学习率
scheduler = torch.optim.lr_scheduler.MultiStepLR(
    optimizer, milestones=[50, 125, 160], gamma=0.5)

# 3. ExponentialLR: 指数衰减
# 每个 epoch 学习率乘以 gamma
scheduler = torch.optim.lr_scheduler.ExponentialLR(optimizer, gamma=0.95)
```

### 6.2 使用方式

```python
for epoch in range(epochs):
    # ... 训练代码 ...

    # 每个 epoch 结束后更新学习率
    scheduler.step()
    print(f"当前学习率: {scheduler.get_last_lr()}")
```

## 七、正则化 - Dropout 随机失活

### 7.1 Dropout 原理

Dropout 是一种**防止过拟合**的正则化技术。在训练过程中，随机将部分神经元的输出置为 0，迫使网络学习更鲁棒的特征。

### 7.2 代码示例

```python
import torch

input_data = torch.randint(1, 10, size=(1, 4), dtype=torch.float32)
linear1 = torch.nn.Linear(4, 5)

# 创建 Dropout 层，p=0.2 表示 20% 的元素被随机置零
dropout = torch.nn.Dropout(p=0.2)

# 前向传播
x = linear1(input_data)
x = torch.tanh(x)

# Dropout 随机失活
x = dropout(x)  # 训练时激活，测试时自动关闭
```

### 7.3 注意事项

- `model.train()` 模式：Dropout 生效
- `model.eval()` 模式：Dropout 自动关闭
- 常用 p 值：0.1 ~ 0.5

## 八、批量归一化（Batch Normalization）

### 8.1 三种 BatchNorm 类型

| 类型 | 适用数据 | 输入形状 |
|------|---------|---------|
| BatchNorm1d | 一维数据（文本） | (N, num_features) |
| BatchNorm2d | 二维数据（图片） | (N, C, H, W) |
| BatchNorm3d | 三维数据（视频） | (N, C, D, H, W) |

### 8.2 代码示例

```python
# BatchNorm2d - 图片处理
input_2d = torch.randn(size=(1, 2, 3, 4))  # 1张图, 2通道, 3×4
bn2d = torch.nn.BatchNorm2d(
    num_features=2,     # 通道数
    eps=1e-5,           # 防止除零的小常数
    momentum=0.1,       # 动量参数
    affine=True         # 可学习的缩放(γ)和偏移(β)参数
)
output = bn2d(input_2d)

# BatchNorm1d - 文本/特征处理
input_1d = torch.randn(size=(2, 2))  # 2个样本, 2个特征
linear1 = torch.nn.Linear(2, 3)
bn1d = torch.nn.BatchNorm1d(num_features=3)
output = bn1d(linear1(input_1d))
```

### 8.3 BN 的作用

- 加速训练收敛
- 允许使用较大学习率
- 有一定的正则化效果
- 减少对参数初始化的敏感性

## 九、完整案例：ANN 手机价格分类

### 9.1 项目概述

使用人工神经网络对手机价格进行多分类预测。完整流程：数据读取 → 预处理 → 模型搭建 → 训练 → 预测评估。

### 9.2 数据准备

```python
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from torch.utils.data import TensorDataset

def create_data():
    # 1. 读取 CSV 文件
    data_df = pd.read_csv("data/手机价格预测.csv", encoding="UTF-8")

    # 2. 分离特征和目标
    x = data_df.iloc[:, :-1]  # 特征数据
    y = data_df.iloc[:, -1]   # 目标值（价格类别）

    # 3. 划分训练集和测试集
    x_train, x_test, y_train, y_test = train_test_split(

    # 5. 转换为 Tensor
    train_dataset = TensorDataset(
        torch.tensor(x_train, dtype=torch.float32),
        torch.tensor(y_train.values, dtype=torch.int64))  # 分类目标必须是 int64
    test_dataset = TensorDataset(
        torch.tensor(x_test, dtype=torch.float32),
        torch.tensor(y_test.values, dtype=torch.int64))

```

**数据处理流程**：`CSV文件 → DataFrame → Tensor → TensorDataset → DataLoader`

### 9.3 模型搭建

```python
class PhoneANNModel(nn.Module):
    def __init__(self, features, targets):
        super().__init__()
        self.linear1 = nn.Linear(features, 128)    # 隐藏层1
        self.linear2 = nn.Linear(128, 256)          # 隐藏层2
        self.output = nn.Linear(256, targets)       # 输出层

    def forward(self, x):
        x = torch.relu(self.linear1(x))
        x = torch.relu(self.linear2(x))
        # 注意：输出层不加 Softmax！因为 CrossEntropyLoss 内部已包含
        x = self.output(x)
        return x
```

### 9.4 模型训练

```python
def train_model(train_dataset, features, targets):
    torch.manual_seed(815)
    dataloader = DataLoader(train_dataset, batch_size=8, shuffle=True)

    model = PhoneANNModel(features, targets)
    criterion = nn.CrossEntropyLoss()
    optimizer = torch.optim.Adam(model.parameters(), lr=1e-3, betas=(0.9, 0.99))

    epochs = 50
    for epoch in range(epochs):
        total_loss = 0.0
        total_sample_num = 0

        for x_train, y_train in dataloader:
            model.train()  # 训练模式（Dropout 生效）
            y_pred = model(x_train)
            loss = criterion(y_pred, y_train)

            total_loss += loss.item() * len(x_train)
            total_sample_num += len(x_train)

            optimizer.zero_grad()
            loss.sum().backward()
            optimizer.step()

        avg_loss = total_loss / total_sample_num
        print(f"第{epoch+1}轮，平均损失: {avg_loss}")

    # 保存模型
    torch.save(model.state_dict(), "model/phone_ann.pt")
```

### 9.5 模型预测与评估

```python
def predict_model(test_dataset, features, targets):
    model = PhoneANNModel(features, targets)
    model.load_state_dict(torch.load("model/phone_ann.pt"))

    dataloader = DataLoader(test_dataset, batch_size=3, shuffle=False)

    correct_count = 0
    for x_test, y_test in dataloader:
        model.eval()  # 评估模式（Dropout 关闭）
        y_pred = model(x_test)
        # 使用 argmax 获取预测类别
        y_pred_target = torch.argmax(y_pred, dim=1)
        correct_count += (y_pred_target == y_test).sum()

    acc = correct_count.item() / len(test_dataset)
    print(f"预测准确率: {acc}")
```

### 9.6 关键知识点总结

1. **数据类型**：特征必须是 `float32`，分类目标必须是 `int64`
2. **CrossEntropyLoss**：内置 Softmax，模型输出层不需要再加 Softmax
3. **train()/eval() 模式**：控制 Dropout 和 BatchNorm 的行为
4. **模型保存/加载**：`torch.save(model.state_dict(), path)` / `model.load_state_dict(torch.load(path))`
5. **argmax**：从模型输出中取最大值的索引作为预测类别
